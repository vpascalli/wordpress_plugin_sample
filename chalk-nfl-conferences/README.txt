== Description ==

Chalk NFL Team plugin, pulls team list from rest api for display

== Installation ==


1. Upload `chalk-nfl-conferences.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add chalk_api_key through plugin settings page available through admin dashboard
3. Place `[chalk_teamlist]` shortcode in your page content
